# QuickReceipt

## Overview
QuickReceipt is a web application for managing receipts, designed for freelancers and small businesses.

## Features
- Dashboard with quick receipt creation
- Client management (CRUD)
- Receipt creation and PDF generation
- Company settings with logo upload
- Multilingual support (FR, EN, AR with RTL)
- WhatsApp and email sharing

## Tech Stack
- Backend: Python Flask
- Frontend: HTML, Tailwind CSS
- PDF: ReportLab
- i18n: Custom solution

## Running
```bash
python app.py
```
